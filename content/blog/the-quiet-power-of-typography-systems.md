Why type is the most underrated leverage point in product design, and how to build a type system that ages well.

## Type Carries the Voice

Before a single word is read, the typeface has already said something. Calm or loud. Formal or warm. Modern or traditional. Choosing the right typeface is one of the highest-leverage decisions in a brand.

## Limit the System

The best product type systems use two typefaces at most. One for display, one for body. Add a monospace for code if needed. Every additional family is a tax on consistency.

## Build a Scale, Not a Pile

A type scale with five to seven sizes covers almost every UI need. Resist the urge to invent new sizes for special cases. The same scale should work in marketing and product surfaces.

- Display sizes for headlines
- Body sizes for paragraphs
- Caption size for metadata
- A single mono size for code

## Line Height Is Underrated

Most readability problems are line height problems in disguise. Tight headlines, generous body line height, and consistent vertical rhythm make text feel calm and expensive.

## Test on Real Devices

Type that looks beautiful on a 5K display can be unreadable on a budget Android phone. Always test the system on the lowest-end device your audience uses.
